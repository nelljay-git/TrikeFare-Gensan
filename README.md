# TrikeFare-Gensan

TrikeFare Gensan is a web-based fare estimation and route reference system designed for commuters in General Santos City. The application provides users with transparent and standardized tricycle fare computations based on the local fare matrix, helping reduce inconsistencies in fare estimation during daily travel.

The system is built to support commuter decision-making by offering clear fare breakdowns according to distance and applicable rate rules (e.g., base fare for the first kilometers and incremental charges for succeeding distances). It aims to improve accessibility, fairness, and transparency in urban transport within the city.

In addition to fare estimation, TrikeFare Gensan may also include route guidance and location-based features to assist users in understanding optimal travel paths across common destinations in General Santos City.

Overall, the project serves as a practical tool for enhancing commuter experience and promoting standardized fare awareness in local tricycle transportation systems.
